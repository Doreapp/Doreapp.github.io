text-checker

Python library to check a text spelling using reverso

Posted on

This personal project is a command line tool, built with python, allowing the user to spell check its text, using online tools such as Reverso.

From a text, the tool find mistakes and suggest corrections.

A continuous integration pipeline is setup, which checks the structure of the code using black, pylint and isort.
The tool uses requests library to interact with Reverso API.
